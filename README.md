# RECALL STARS prototype

『リコール・スターズ』は、未知の星域へ転送された5人小隊を率いるPCブラウザ向けの**3D縦スクロール・小隊サバイバルシューティング**です。

画面手前の小隊を動かし、奥から接近する敵性反応を迎撃します。一定時間を生き延び、星域ごとに異なるボスを撃破すると帰還できます。HPが0になったメンバーはその任務中に復帰せず、全員が戦線離脱するとゲームオーバーです。

## 起動方法

ビルドやパッケージインストールは不要です。

1. VS Codeでこのフォルダーを開く
2. Live Server拡張機能で `index.html` を開く
3. タイトル画面の `MISSION START` を押す

Three.jsをCDNから読み込むため、初回起動時はインターネット接続が必要です。`file://` で直接開かず、Live ServerなどのローカルHTTPサーバーを使用してください。

## 操作方法

| 操作 | 1P | 2P |
|---|---|---|
| 移動 | WASD / 矢印キー | IJKL |
| ショット | Space | Enter |
| ダッシュ | Shift | Right Shift |

- `P`：2P参加 / 離脱
- `R`：現在の任務をリスタート
- `M`：新しいランダム任務を生成
- `B`：背景生成パターンを切り替え
- `H`：操作説明を表示 / 非表示

## 実装済み機能

- Three.jsによる固定カメラ式3D縦スクロールシューティング
- 候補15人から毎回ランダムに選ばれる5人小隊
- 攻撃・慎重・救援・収集・臆病の5種類の擬似AI
- 1P操作と、途中参加できるローカル2P同時プレイ
- HP、戦線離脱、粒子化、離脱ログ、全滅ゲームオーバー
- 直進・追尾・横揺れ・自爆・砲台の5種類の敵
- 発光弾、軌跡、ヒット粒子、弾数・粒子数上限
- 小惑星、結晶、回復・エネルギーアイテムのフィールド生成
- 複数攻撃、後半強化、専用HPバーを持つランダム名ボス
- 星域名、任務、敵性起源、異常ルール、短い物語の自動生成
- 帰還リザルト、全滅記録画面、Web Audio APIの軽量効果音
- リサイズ対応と描画オブジェクト数の制限

## ランダム生成AI風宇宙背景

本物の生成AIや外部画像は使わず、Three.jsのプリミティブ、Points、ライン、ワイヤーフレーム、CanvasTextureを組み合わせ、毎回「宇宙がその場で生成されている」ように見せています。

背景モデルは、星雲型・ワイヤー惑星型・ノイズ宇宙型・小惑星帯型・AI生成途中型の5種類です。星の密度、色相、巨大構造物、星雲、座標線などはミッションごとに変化し、`B` キーで任務中にも再構成できます。

## 作品上の位置づけ

突然集められた人々が理不尽な任務へ送られ、仲間を失いながら生還を目指す構造には『GANTZ』からの着想があります。一方で、固有の名称、黒い球、点数制度、スーツ、武器、敵名称、演出、用語は使用せず、独立した世界観・ルール・ビジュアルとして制作しています。
